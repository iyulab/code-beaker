- **Docusaurus docs**: `lowercase-with-dashes.md` (e.g., `architecture.md`)
- **Phase reports**: `PHASE##_NAME_COMPLETE.md` (e.g., `PHASE11_PRODUCTION_HARDENING_COMPLETE.md`)
